## Projet Android TD1

[Voir sujet TD1](https://github.com/Lion3000/Projet-Android/blob/master/android_td1.pdf)


### Technologies utilisées:

* [Android Studio](https://developer.android.com/studio/index.html#win-bundle)
* [Kotlin](https://kotlinlang.org/)
* [Git](http://git-scm.com/)
	
### Déploiement du projet

* Créer un compte sur [Bitbucket](https://developer.android.com/studio/index.html#win-bundle) (si nécessaire)
* Cloner le projet avec la ligne de commande

```
git clone https://bitbucket.org/lpsilidseandroid/android-hello-world.git TD1
```
	
* Ouvrir Android Studio
* Ouvrir le dossier TD1
